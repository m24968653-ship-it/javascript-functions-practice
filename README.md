# JavaScript Functions Practice

This is a simple project to practice fundamental JavaScript functions.  
It demonstrates the use of **regular functions**, **arrow functions**, **parameters**, **return values**, and **console output**.

---

## Features

- **Calculate age in days**  
  Convert a person's age in years to total days.

- **Calculate final price with tax and discount**  
  Compute the final price of an item after adding tax and subtracting optional discount.

- **Calculate average score**  
  Compute the average of three subject scores using an arrow function.

- **Check if a person is an adult**  
  Returns `true` if age is 18 or older, otherwise `false`.
